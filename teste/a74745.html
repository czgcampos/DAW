Resumo

Este teste é composto por dois exercícios, o primeiro a criação de uma aplicação que faz o encaminhamento de dados vindos de uma coleção de compositores, sendo que esta foi criada através de um ficheiro JSON, previamente tratado.

O segundo exercício consite no desenvolvimento de uma aplicação que serve apenas como interface, para o projeto CLAV, que é uma parceria da Universidade do Minho com a Direção Geral do Livro. Esta interface mostra informação sobre os processo que estão armazenados no catálogo dos processos da administração pública portuguesa.
